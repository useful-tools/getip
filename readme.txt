Оповещение о смене IP адреса администрируемой станции
1. Через свой сервер с php скриптом (http://<ownserver.com>/ip.php)
2. Отправка почтой данных ipconfig (blat)
3. Триггер для DDNS (http://freedns.afraid.org/)

ip.php - серверная часть
getip.bat - клиентская часть, может выполняться в диспетчере задач клиентской станции 

wget.exe, blat.exe и скрипт ddns в комплект не входят

Перед использованием настроить свои переменные в заголовках скриптов:
 адрес хоста, адрес почты для репортов, настройки локальных подключений к smtp

Известные проблемы:
 если вывод ipconfig очень большой, может не поместиться в переменную datatext, 
 скрипт выдаст ошшибку:
  Слишком длинная входная строка.
  Ошибка в синтаксисе команды.
